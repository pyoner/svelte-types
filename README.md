# Typescript definitions for Svelte v3

## Install
```bash
npm i -D @pyoner/svelte-types
```

## How to use?

Add to your `tsconfig.json`
```javascript
{
  "compilerOptions": {
    // ... your config
    "types": ["@pyoner/svelte-types"]
  }
}
```
